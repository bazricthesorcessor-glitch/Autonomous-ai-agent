# 🧠 Avril — Autonomous AI Agent

> A UI-first, multi-agent AI system that can **see**, **reason**, and **act** on a computer like a human.

---

## ⚡ Overview

Avril is an experimental autonomous AI agent designed to interact with real-world computer environments using:

* 👁 **Screen perception (vision + OCR)**
* 🧠 **Executive reasoning (multi-agent orchestration)**
* 🖱 **UI automation (mouse + keyboard control)**
* 💾 **Structured memory system**
* 🔒 **Safety-first execution model**

Unlike traditional chatbots, Avril operates **directly on the interface layer**, enabling real task execution.
